# DIY Outfits 🎀 A fun dress-up styling game — **made by Claree**. Style a girl or a boy, mix tops, bottoms, shoes, hairstyles, patterns and accessories, save your looks to a wardrobe, then put them on a judged runway with different poses (including **Pose 28 🔥**). Earn coins, and the colour theme changes every 2 rounds. It's a single self-contained file — no build step, no dependencies. ## Play it locally Just double-click **`index.html`**, or open it in any web browser. ## Put it on GitHub 1. Create a new repository on GitHub (e.g. `diy-outfits`). 2. Upload the contents of this folder (`index.html`, `README.md`, `render.yaml`) to the repo. The easiest way on the website: **Add file → Upload files → drag everything in → Commit changes.** ## Deploy on Render (free static site) **Option A — the simple way (no config needed):** 1. Go to and click **New → Static Site**. 2. Connect your GitHub account and pick the `diy-outfits` repo. 3. Set: - **Build Command:** *(leave empty)* - **Publish Directory:** `.` 4. Click **Create Static Site**. Render gives you a live link in a minute. 🎉 **Option B — using the included blueprint:** This folder includes `render.yaml`, so you can instead choose **New → Blueprint**, pick the repo, and Render will configure the static site automatically. ## Editing the game Open `index.html` in any text editor.
